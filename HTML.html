HTML (HyperText Markup Language)

HTML adalah bahasa markah yang digunakan untuk membuat struktur dasar sebuah halaman web. HTML terdiri dari elemen-elemen yang terdiri dari tag, atribut, dan konten. HTML adalah fondasi dari semua halaman web di internet.

Struktur Dasar HTML

Berikut adalah struktur dasar dari sebuah dokumen HTML:

            <!DOCTYPE html>
            <html>
                <head>
                    <meta charset="UTF-8">
                    <title>Judul Halaman</title>
                </head>
                <body>
                    <h1>Selamat datang di website saya!</h1>
                    <p>Ini adalah paragraf pertama.</p>
                </body>
            </html>

Tag HTML Umum

Beberapa tag HTML yang sering digunakan dalam pembuatan halaman web:

  • <html>: Tag root yang menandakan dokumen HTML.
  • <head>: Tag untuk metadata halaman (seperti judul dan pengaturan karakter).
  • <body>: Tag untuk konten halaman yang terlihat oleh pengguna.
  • <h1> - <h6>: Tag untuk judul, dengan <h1> sebagai yang paling besar dan <h6> sebagai yang terkecil.
  • <p>: Tag untuk paragraf.
  • <a>: Tag untuk membuat tautan (link).
  • <img>: Tag untuk menampilkan gambar.

Contoh Penerapan HTML

Berikut adalah contoh sederhana penggunaan HTML untuk menampilkan teks dan gambar:

            <html>
                <head>
                    <title>Contoh HTML</title>
                </head>
                <body>
                    <h1>Ini adalah judul utama</h1>
                    <p>Ini adalah sebuah paragraf dalam HTML</p>
                    <img src="gambar.jpg" alt="Gambar Contoh">
                </body>
            </html>
