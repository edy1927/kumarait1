CSS (Cascading Style Sheets)

CSS adalah bahasa yang digunakan untuk mendesain dan mengatur tampilan elemen-elemen HTML dalam sebuah halaman web. CSS memungkinkan kita untuk mengubah warna, font, layout, dan banyak aspek lainnya dari sebuah halaman.

Dasar-Dasar CSS

CSS mengatur tampilan elemen-elemen HTML menggunakan selektor dan deklarasi. Berikut adalah struktur dasar dari CSS:

            selector {
                property: value;
            }

Contoh sederhana CSS:

            body {
                background-color: lightblue;
            }
            h1 {
                color: darkblue;
            }

CSS juga memiliki beberapa jenis pengaplikasian:

  • Inline CSS: CSS ditulis langsung dalam elemen HTML menggunakan atribut style.
  • Internal CSS: CSS diletakkan dalam tag <style> di bagian <head> pada dokumen HTML.
  • External CSS: CSS ditulis dalam file terpisah dengan ekstensi .css dan di-link ke file HTML menggunakan tag <link>.

Contoh Penerapan CSS

Berikut adalah contoh penerapan internal CSS dalam halaman HTML:

            <style>
                body {
                    background-color: #f0f8ff;
                }
                h1 {
                    color: #2e4053;
                }
            </style>
